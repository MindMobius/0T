# 《魔轨列车》项目概念企划 (Project Magic Rail)

> **一句话介绍：**
> 这是一款**多人联机、以移动列车构筑为核心**的生存冒险游戏。在这个被“魔星”周期性辐照的世界里，玩家驾驶着可高度自定义的魔轨列车，在五大势力间穿梭、贸易、战斗，体验**“移动基地+移动塔防+自动化生存”**的独特快感。

---

## 0. 设计宣言（我们的核心价值观）
*在讨论具体机制前，我们需要达成共识，这不仅仅是“又一个生存游戏”，我们要解决以下痛点：*

1.  **拒绝无聊的数值堆砌：** 我们不要“木剑换铁剑，铁剑换钻石剑”的无趣升级。**科技树是水平发展的**，不同势力的科技代表不同的生态位（生物科技 vs 机械工业 vs 反重力），它们是玩法的改变，而非单纯的数值碾压。
2.  **基地的意义在于移动：** 玩家不需要反复搬家，**列车就是家**。它集成了仓储、制造、防御。我们鼓励探索，而不是被“回家存东西”束缚。
3.  **自动化解放双手：** 拒绝重复砍树挖矿。琐碎的工作交给AI、魔像、无人机。玩家的时间应该花在**构筑、策略、战斗和探险**上。
4.  **有尊严的塔防：** 结合塔防的爽感与动作游戏的灵活性。你可以坐在车里指挥自动炮台（塔防），也可以穿上外骨骼跳下列车冲锋（ARPG）。打不过？**开车跑！**
5.  **软生存，强体验：** 生存不是为了恶心玩家（不死于饥饿），而是为了驱动探索。死亡不应清零，积累必须保留。我们要的是挑战Boss和恶劣环境的刺激，而不是重头再来的挫败感。

---

## 1. 世界观概要：追逐安全区的破碎世界

**核心矛盾：**
一颗名为**“魔星”**的天体每360天环绕地球一周。它带来的**“魔能辐照”**会导致生物异变（魔物）、植物疯长（魔植）、矿物充能（魔晶）。

**人类的生存方式：**
人类无法在这个星球上定居，必须根据魔星的位置进行**周期性迁徙**或**形态转换**。魔能既是灾难，也是科技的源头。

**五大势力（独特的科技树与生态）：**

| 势力名称 | 核心特征 | 应对辐照手段 | 特殊科技/产出 |
| :--- | :--- | :--- | :--- |
| **逐光号** | **环球游牧商会** | 永远在移动，追逐背光面 | **光晶科技**：净化、护盾、通用货币。拥有全球轨道网络。 |
| **浮空城** | **赛博天空都市** | 城市底朝天翻转，物理遮挡 | **反重力引擎**：高机动性、飞行模块、尖端机器人。 |
| **绿盟** | **生物共生群落** | 控制巨型植物覆盖建筑 | **生物魔控**：异变生物兵器、再生医疗、特殊食品。 |
| **熔核区** | **地下重工帝国** | 全员下潜至地下掩体 | **魔核工业**：超重型装甲、热能武器、钻探机械。 |
| **岛联** | **海洋潜行联邦** | 潜入深海，利用海水隔绝 | **海魔晶**：反射涂层、水下推进、光学迷彩。 |

## 2. 核心玩法循环

### **Step 1: 构筑（The Build）**
*   **移动堡垒：** 游戏的核心是你的**列车**。
    *   **模块化建造：** 标准车厢搭载 `2x3` 体素网格。你需要像玩积木一样拼装：
        *   *功能舱：* 引擎、驾驶室、生活区、合成台。
        *   *战斗舱：* 自动炮塔、护盾发生器、无人机蜂巢。
        *   *资源舱：* 矿物提炼机、全自动分拣货仓。
    *   **子母机系统：** 模块不仅能装在车上，还能装在**“矿骡”**（小型载具）上脱离列车独立作业，或者部署在地面变成临时哨站。

### **Step 2: 探索与贸易（The Journey）**
*   **跑商系统：** 利用信息差赚钱。从“熔核区”低价买入重型装甲，运到“绿盟”高价卖出，换取那里的珍稀草药。
*   **以太场（Ether Field）：** 玩家自带的超能力。
    *   *功能：* 类似穿墙透视/黑客扫描。可以远程操控设备、获取情报、扫描矿脉。
    *   *AI伴侣“巴啦”：* 你的战术副官，负责整理信息和吐槽。

### **Step 3: 战斗与生存（The Fight）**
*   **双层级战斗：**
    *   *微观（人）：* 玩家身穿魔能外骨骼，手持兵器，下车进入遗迹/洞穴探索，对抗精英怪和Boss。
    *   *宏观（车）：* 面对兽潮（魔物大军），玩家驾驶列车进行移动塔防。列车在轨道上飞驰，两侧炮火轰鸣，你可以亲自操控主炮，也可以指挥AI自动索敌。
*   **辐照机制：** 这是一个类似SAN值/辐射值的系统。高辐照区=高收益+高风险。必须依靠列车的净化装置或特定药剂维持。

## 3. 游戏特色系统（卖点）

### **A. 科技树：横向专精 (Horizontal Progression)**
没有绝对的“最强装备”。
*   你想玩**召唤流**？去刷**绿盟**声望，把车厢改成生物孵化室，列车行驶过的地方长满食人花帮你打怪。
*   你想玩**高机动**？去搞**浮空城**科技，给列车装上喷气引擎，遇到断轨直接飞过去。
*   你想玩**硬硬硬**？**熔核区**的超重甲让你的车变成推土机，直接撞碎魔物。

### **B. 自动化网络**
*   后期不需要玩家自己挖矿。在富矿区部署自动采掘机，设置好无人机运输线，列车路过时自动吸入物资。
*   只要你规划得好，那就是一个**“长在轮子上的异星工厂”**。

### **C. 角色定位（联机）**
*   **独狼玩家：** 打造全能型列车，带足AI机器人。
*   **联机车队：**
    *   玩家A：**车长**，负责驾驶、护盾管理、路线规划。
    *   玩家B：**炮手/突击队**，负责操作主炮，或下车清理路障。
    *   玩家C：**后勤/工程师**，负责车内资源调度、弹药制造、紧急抢修。

## 4. 为什么这个游戏好玩？（寻找知音）

*   **如果你喜欢《泰拉瑞亚》：** 我们有丰富的Boss战、地下城探索和基于装备驱动的成长。
*   **如果你喜欢《异星工厂》/《Mindustry》：** 我们有资源管理和自动化防御流水线，但没有那么硬核烧脑，更偏向爽快。
*   **如果你喜欢《Rafty》/《虚空列车》：** 我们有移动基地的浪漫，但我们的世界更广阔，有明确的文明和势力交互，不只是孤独的漂流。

---

## 5. 待讨论的开放性问题（我们需要一起做的事）

目前这些只是核心设想，以下内容需要我们共同填充：
*   **游戏视角：** 是第一人称，还是第三人称？
*   **轨道建设：** 玩家是可以自由铺设轨道去往荒野，还是主要依赖势力间的固有轨道？（这决定了地图自由度）
*   **列车物理：** 这是一个基于网格的魔法列车，还是需要考虑转弯半径和物理碰撞的拟真列车？
